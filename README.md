# coish.net

Static splash / marketing site for [coish.net](https://www.coish.net).

## Deploy (Vercel)

1. Import this repo in Vercel (Framework preset: **Other**, no build command).
2. Add domain **coish.net** (and `www.coish.net` if used) in Project → Settings → Domains.

## Local preview

Open `index.html` in a browser, or:

```bash
npx serve .
```
